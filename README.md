# wireshark-analizlerim
Wireshark analiz raporlarım tammen el ile yazılmıştır.Sürekli olarak güncel tutmaya çalışıyorum.
## What This Project Does
Analyzes DNS logs to detect beaconing behavior commonly associated with malware C2 communication.

## Why It Matters
Early detection of beaconing activity helps prevent data exfiltration and persistent access.

## Tools Used
- Python
- Wireshark
- MITRE ATT&CK Framework
